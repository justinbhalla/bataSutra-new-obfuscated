Latest releases

2025-09-22
Sector Heatmap — Feature

New filter chips (Market-cap, Beta, Earnings) with sticky state. Improves analyst workflow on mobile.

Data pipelines — Perf

Daily backfill time reduced by ~34%; retries are now jittered to avoid exchange throttle windows.

IPO Tracker tooltip — Fix

Resolved a Safari hover flicker when moving across the pricing band badge.

2025-09-18
Cookies banner — Deprecation

Removed legacy consent SDK; moved to lightweight, zero-tracking notice.

Commodities dashboard — Data

Added LME aluminum inventories, RBI reference FX snapshot, and MCX settlement notes.

2025-09-12
Accessibility — Fix

Contrast and focus styles improved for nav chips; landmarks labeled.

Highlights

Mobile polish
Sticky filters, better tap targets, contextual tooltips on data tables.
Reliability
Fewer red deploys; faster fallbacks when an upstream is late.
New datasets
Global benchmarks and RBI daily ref FX added to more dashboards.

Versioning & channels

  • SemVer-ish We group UI & data changes by date; breaking API changes (if any) are versioned separately.
  • Channels Canary → Beta → Stable. High-risk changes soak on canary 24–72h.
  • Flags Big features ship behind flags and are rolled out progressively.

What counts as breaking?

Public URL or contract changes, removed metrics, or altered semantics without alias/migration path.
