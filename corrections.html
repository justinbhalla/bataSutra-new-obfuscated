How to submit a correction

  1. Include the URL, the specific passage/chart, and the suggested fix with sources.
  2. Attach relevant documents, data screenshots, or public links as evidence.

Response times

  • Acknowledge: within 2 business days
  • Clarifications: as needed via email
  • Resolution: varies by complexity and source checks

Review & decision workflow

  • Triage by editor for scope and potential impact.
  • Verification using documents, data, and subject-matter checks.
  • Decision: correction (material), clarification (minor), or no change with rationale.
  • Update story/tool and add visible correction/clarification note if applicable.

Correction labels

Label When used Note text
Correction Material factual error changed “Correction (YYYY-MM-DD): … what changed …”
Clarification Added context without changing core facts “Update (YYYY-MM-DD): … clarified …”
Editor’s note Method/base-year/source revisions “Editor’s note (YYYY-MM-DD): … revision …”

Where notes appear

  • Story footers or near the affected passage
  • Chart/tool footnotes or “Methodology” box

Transparency & archive

  • We keep a public note on material corrections in-line or in the footer.
  • For major data/method changes, we link to Methodology or a short explainer.
  • We archive prior outputs when comparisons matter (e.g., base-year changes).
